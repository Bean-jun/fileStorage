# 文件对象存储

### 一、使用方式

#### 1 常规方式
##### 1.1 安装依赖
```shell
pip install -r requirements.txt
```
##### 1.2 启动实例
```shell
python3 app.py
```
##### 1.3 查看api文档
```shell
http://127.0.0.1:8080/docs
```

#### 2. Docker-compose

##### 2.1 直接run起
```shell
docker-compose up
```
##### 2.2 查看api文档
```shell
http://127.0.0.1:8080/docs
```
### 二、目前功能

1. 单文件上传
2. 多文件上传
3. 文件下载
4. 文件删除
